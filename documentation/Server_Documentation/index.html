Smashville Server Documentation

Description

Smashville is a tribute game to the Nintendo game Super Smash Bros with inspirations from other classic games, such as Street Fighter 2. Players will be able to run a local version of our game, or connect to a hosted server, provided by Node.js and Socket.io. Once connected players can either host an open session for others to join, or connect to an already existing session.
